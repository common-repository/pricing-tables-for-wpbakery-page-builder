Fully Customizable Responsive Pricing Tables for Visual Composer.

== Description ==

The best way to showcase your websites products & services inside of WPBakery Page Builder (Visual Composer). Choose from 20+ customizable Price Table layouts we created just for your needs. This pack is truly awesome and unique in its design and usability.

<a href="http://demo.webdevocean.com/">See DEMO Here</a><br>

Note: This plugin requires a href="http://codecanyon.net/item/visual-composer-page-builder-for-wordpress/242431?ref=labibahmed" target="_blank">Visual Composer</a> Plugin.

== Installation ==
1. Go to plugins in your dashboard and select 'add new'
2. Search for 'Pricing Tables For WPBakery Page Builder', install and activate it
3. Go to post and page and select element naming with "Hover Effects With Lightbox - VC Extension".
4. Choose desired options and save settings.
5. View page or post.
